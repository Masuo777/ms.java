## 将本地代码与远程仓库绑定



## 将本地代码提交到远程仓库步骤

1. 将待更新文件添加到本地仓库

    ```bash
    git add .
    ```

    

2. 提交本地仓库（携带注释信息）

    ```bash
    git commit -m "注释信息"
    ```

    

3. 拉取所在分支的远程仓库

    ```bash
    git pull origin <分支名>
    ```

    

4. 将本地仓库推到远程仓库

    ```bash
    git push origin <f>
    ```
